## Contents

T commands fall into several categories:

- I/O verbs
    - from
    - write
- Row verbs
    - keep, drop
    - rename, alias
    - select
    - derive
    - first, last, sample
    - cast
- Table verbs
    - sort
    - groupby
    - join
    - union
- Display verbs
    - show
    - inspect
- Stack verbs
    - clear
    - pop
    - swap
    - reverse
    - rotate
- Miscellaneous verbs
    - duplicate
    - history
